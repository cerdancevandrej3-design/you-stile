## ЛЕТО 2026 — ГОТОВЫЕ ОБРАЗЫ И РЕКОМЕНДАЦИИ

### Летние образы (5 категорий):

**1. Пляжный кэжуал (Beach Casual)**
- Льняные шорты + хлопковая футболка-поло + сандалии
- Соломенная шляпа + хлопковая рубашка oversize + лёгкие брюки чинос
- Хлопковое платье-макси + эспадрильи + плетёная сумка
- Льняной комбинезон + кожаные сандалии + очки авиаторы

**2. Городской летний (Urban Summer)**
- Льняные брюки + шёлковая рубашка + белые кожаные кроссовки
- Хлопковые чинос + поло пике + лоферы + соломенный берет
- Лёгкий льняной костюм (брюки + рубашка) + эспадрильи
- Миди-платье в полоску + кожаные сандалии + лёгкая джинсовая куртка

**3. Летний элегантный (Resort Elegance)**
- Шёлковое платье + лёгкие лоферы + минималистичные украшения
- Льняной сарафан + кожаные босоножки на низком каблуке + панама
- Хлопковый костюм светлых тонов + лёгкие дерби + шёлковый шарф
- Брючный костюм в бежевом + топ + эспадрильи на танкетке

**4. Пикник / Выходной (Weekend Picnic)**
- Джинсы + хлопковая рубашка + белые кроссовки + кепка
- Льняные шорты + футболка + сандалии + пляжная сумка
- Платье-миди + кроссовки + джинсовая куртка
- Комбинезон + эспадрильи + соломенная шляпа

**5. Летний вечер (Summer Evening)**
- Шёлковая рубашка + льняные брюки + лоферы + часы
- Лёгкое платье + босоножки + украшения-минимализм
- Костюм светлый + топ + эспадрильи
- Макси-платье + сандалии + клатч

### Летняя палитра цветов:
- Кремовый, экрю, слоновая кость, белый
- Песочный, бежевый, карамель
- Светло-голубой, мятный, нежно-розовый, коралловый
- Сизый, лавандовый, персиковый
- Глубокие акценты: тёмно-синий, бордовый (точечно)

### Летние ткани:
- Лён (главная ткань лета)
- Тонкий хлопок (поплин, пике)
- Шёлк, вискоза, муслин, тенсель
- Летняя шерсть super 120s (тонкая, для вечерних костюмов)

### Летняя обувь:
- Сандалии (минималистичные, гладиаторские, birkenstock-тип)
- Эспадрильи (кожаные, на танкетке)
- Белые кожаные кроссовки (минималистичный дизайн)
- Лёгкие лоферы (босиком или с тонкими носками)
- Открытые дерби из перфорированной кожи
- Босоножки на низком каблуке / танкетке

### Летние аксессуары:
- Соломенные шляпы (панама, федора, пальма)
- Льняные бейсболки
- Плетёные сумки (рафия, солома)
- Холщовые / хлопковые сумки-тоут
- Очки (авиаторы, wayfarer, clubmaster)
- Тонкие кожаные ремни светлых тонов
- Украшения: золото, серебро, натуральные камни
